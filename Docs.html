99 Nights | Bring Items Script

Описание:

Этот скрипт позволяет автоматически собирать предметы из папки workspace.Items в игре 99 Nights. Работает через RemoteEvent RequestStartDraggingItem.

Пример полного списка предметов:

local AllItems = {
    "Old Sack","Good Sack","Infernal Sack","Giant Sack","Admin Sack",
    "Old Axe","Good Axe","Ice Axe","Strong Axe","Chainsaw","Admin Axe",
    "Old Rod","Good Rod","Strong Rod","Old Taming Flute","Good Taming Flute","Strong Taming Flute",
    "Carrot","Corn","Pumpkin","Berry","Apple","Morsel","Cooked Morsel",
    "Steak","Cooked Steak","Ribs","Cooked Ribs","Cake","Chili","Stew","Hearty Stew",
    "Meat Sandwich","Seafood Chowder","Steak Dinner","Pumpkin Soup","BBQ Ribs",
    "Carrot Cake","Jar o' Jelly","Candy Apple","Candy Corn","Pumpkin Pie","Cotton Candy"
    -- (и остальные предметы)
}

Bring Items Скрипт:

local Players = game:GetService("Players")
local ReplicatedStorage = game:GetService("ReplicatedStorage")
local player = Players.LocalPlayer
local Event = ReplicatedStorage:WaitForChild("RemoteEvents"):WaitForChild("RequestStartDraggingItem")

local AllItems = { "Old Sack","Good Sack","Infernal Sack","Giant Sack","Admin Sack", ... } -- полный список

local root = player.Character and player.Character:FindFirstChild("HumanoidRootPart")
if not root then return end

for _, itemName in ipairs(AllItems) do
    local item = workspace.Items:FindFirstChild(itemName)
    if item then
        Event:FireServer(item)
        -- Если предмет модель, телепортируем
        if item:IsA("Model") then
            item:PivotTo(root.CFrame * CFrame.new(0,0,-5))
        else
            item.CFrame = root.CFrame * CFrame.new(0,0,-5)
        end
        task.wait(0.2) -- маленькая пауза
    end
end

Советы:

  • Используйте паузы (task.wait()) между предметами, чтобы сервер не заблокировал события.
  • Для больших списков (>200 предметов) лучше разделять на категории, чтобы UI не лагал.
  • Для моделей используйте PivotTo, чтобы предмет телепортировался корректно.
